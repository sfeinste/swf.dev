Berm Brain

Berm Brain is a simple ride tracking app I built to help remember my bike setup changes. It tracks rides, manages service intervals for multiple bikes, and lets you log setup details like tire pressure and suspension settings for each ride.

The main idea is to compare segment times with different setups. Did that lower tire pressure actually help on the rooty section? Was I running more rebound damping when I cleaned that jump line? Instead of trying to remember what changed between rides, the app keeps track for you.

It includes GPS tracking, automatic segment detection, and comparison tools. You can log tire pressure, suspension PSI, rebound/compression settings, and other setup notes. It also sends reminders for basic maintenance like chain replacements and suspension service.

Built in "America's mountain biking capital", Bentonville AR.

Currently in beta testing on TestFlight. If you'd like to try it out and share feedback, feel free to send me an email.
